# Aurum Números ✨

Plataforma de numerología y lotería dominicana con IA.

## Características

- 🎰 Números de la suerte diarios para todas las loterías DR
- 📅 Horarios de sorteos (Nacional, Leidsa, Real, Loteka, New York)
- 🌙 Interpretación de sueños con IA (OpenAI opcional)
- 💬 Chat con Aurum IA, tu oráculo numerológico
- 👑 Sistema de suscripción Premium
- 🛡️ Panel de administración

## Instalación

```bash
npm install
npm run dev
```

Frontend: http://localhost:5173  
API: http://localhost:3001
